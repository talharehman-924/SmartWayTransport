# 🚀 SmartWay Transfer Dashboard

![SmartWay Hero System](public/output-onlinepngtools.png)

A comprehensive, production-grade Next.js Booking Management Dashboard designed to handle passenger transfer logistics natively.

This web application features strict highly-secure **Role-Based Access Control (RBAC)** completely decoupling standard Member portals from powerful Admin capabilities. Built seamlessly over **Supabase** backend architecture, the dashboard actively handles real-time driver commission assignments, instant multi-format PDF generation caching, and unified dynamic booking scheduling.

## ✨ Key Features

### 🛡️ Unified Login & Authentication
- **Intelligent Gateway**: A single, beautiful responsive login interface that dynamically analyzes the sign-in packet behind the scenes, effectively redirecting superusers into strict Admin routes and directing drivers/members to standard route workflows.
- **`bcrypt.js` Security Validation**: Hardened passwords specifically engineered and instantly hashed before comparing values against backend tables.

### 👥 Role-Based Access Control (RBAC) System
- **Administrator Dashboard (`/admin/dashboard`)**: Full data oversight, custom real-time stats overview panels, secure settings to easily swap your hash configuration (`/admin/settings`), complete driver commission input flows, and database data-table manipulation. The Next.js middleware forcefully kicks random non-admins actively avoiding the URL.
- **Member Dashboard (`/dashboard`)**: Focused and isolated portal purely dedicated strictly towards user workflow without touching infrastructure configurations.

### 🚗 Vehicle Booking & Driver Assignment Flow
- Easily log customer names, location logistics, customized transfer luggage parameters, dynamic vehicle tracking, and package values all within seconds.
- Integrated modal to immediately select drivers strictly from the secure SQL driver pool list. Set the dynamic **Current SAR Commission Rate**, instantly binding them directly to an active booking!

### 🧾 Professional Ticket Document Generation
- Fully functional custom-scaled automatic `html2canvas` mapping! Clicking generation creates absolute 1:1 perfectly rendered layout ratios from real-time database JSON blocks strictly converted to styled standalone **Voucher PDFs**.
- Auto-generates independent **Driver Vouchers** vs unique **Passenger Vouchers** sequentially numbered automatically counting parallel to global DB ID values matching internal tracking mechanisms perfectly!

## 📦 Local Installation & Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/talharehman-924/SmartWayTransfer.git
   cd SmartWayTransfer
   ```

2. **Install Node Utilities**:
   Ensure you have Node installed, then fetch package dependencies (`jspdf`, `html2canvas`, `bcryptjs`, `@supabase/supabase-js`, etc).
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   In the root directory, configure your strict connection to your Supabase host. Create a `.env.local` containing:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=YOUR_PROJECT_URL_HERE
   NEXT_PUBLIC_SUPABASE_ANON_KEY=YOUR_SECURE_ANON_KEY_HERE
   SUPABASE_SERVICE_ROLE_KEY=YOUR_SECURE_SERVICE_ROLE_HERE
   ```

4. **Initialize Supabase RBAC Table**:
   Go to your Supabase SQL Editor and ensure the strict single Master Admin table is generated prior to running:
   ```sql
   CREATE TABLE IF NOT EXISTS public.admin_users (
     id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
     email TEXT UNIQUE NOT NULL,
     password_hash TEXT NOT NULL,
     created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
   );

   -- Hardcoded default hash (Password: Admin@12345)
   INSERT INTO public.admin_users (email, password_hash)
   VALUES ('admin@smartway.com', '$2b$10$4uPLw.DidiRtGv/ArSeageaW3avEYoEnsJACvUjEJndOplU6WZIBi')
   ON CONFLICT (email) DO NOTHING;
   ```

5. **Start Development Server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` to interact with your login portal setup.
